## Manual Docker checklist (v0.9.1)

Run after `docker compose -f infra/docker-compose.prod.yml up -d --build`:

- [ ] Register/login via browser at `http://localhost:8080` (hard refresh if cached)
- [ ] Upload `test.txt` → Load all files → download
- [ ] Create public share → open public URL in incognito
- [ ] Admin stats/audit (if admin user seeded)
- [ ] `compose down` then `up -d` → data persists (`pgdata_prod`, `uploads` volumes)
- [ ] Large upload (>1 MB) returns nginx 413 (documented accidental ceiling, not CWE-400)
