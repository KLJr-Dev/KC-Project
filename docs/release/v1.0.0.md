# v1.0.0 — Pentest-Ready Insecure MVP

Tagged from v0.9.5-rc.1 per [STRATEGY.md](../roadmap/STRATEGY.md).

## What's included

- Full auth, files, sharing, ternary RBAC, admin polish
- Pagination, validation, unified errors, request logging
- Persistent audit logs, admin search/stats, health endpoint
- Docker production stack (`infra/docker-compose.prod.yml`)

## Known intentional weaknesses

See [cwe-inventory.md](../security/cwe-inventory.md). Target 60–80 CWEs.

## Deploy

```bash
docker compose -f infra/docker-compose.prod.yml up -d --build
./infra/smoke-test.sh
```
